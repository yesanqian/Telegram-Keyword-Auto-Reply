### 1. 全方位智能匹配
* **多维度识别**：不仅支持**消息内容**匹配，还支持对**用户名**、**文件名**进行关键词或正则扫描，有效拦截“改名党”广告。
* **毫秒级响应**：识别关键词后即刻触发，支持文本、图片及各类多媒体文件答疑。

### 2. 高级风控管理
* **正则精准拦截**：配合 Regex 规则，从源头封杀恶意链接、特定后缀文件及违规 `@` 拉人行为。

### 3. 极简操作与专业管理
* **可视化配置**：支持按钮拖拽逻辑，无需代码基础，新手也能快速上手。
* **配置导入/导出**：支持配置文件的备份与迁移，方便在多个群组间快速复用。
* **代码模式**：为高级用户提供代码编辑模式，支持深度定制复杂的回复逻辑。
